


This project is designed to predict the price of a house based on home sales listing data. 

Data includes price, number of bedrooms and bathrooms, square footage, as well as a home luxury classification, house type classification, and lot size classification. 

These classifications were made through a query to Claude 4-5 using images of the homes taken from REALTOR.com.   

This project was developed with the assistance of Microsoft Copilot, as well as the general project outline from here: https://github.com/Snowflake-Labs/snowflake-demo-notebooks/tree/main/First_Machine_Learning_Project.


I have also included the dataset used in this repository.

The CSV file inlcudes links to the original REALTOR.com listing web pages.
